# @hit/feature-pack-marketing

A HIT feature pack

## Installation

```bash
hit feature add marketing
```

Or manually:
1. Add to `hit.yaml` under `feature_packs:`
2. Add `"@hit/feature-pack-marketing": "github:c0x65o/hit-feature-pack-marketing"` to `package.json`
3. Add `'@hit/feature-pack-marketing'` to `hitPackages` array in `next.config.js`
4. Run `npm install`
5. Run `hit run` to generate routes

## Development

```bash
npm install
npm run build
```
